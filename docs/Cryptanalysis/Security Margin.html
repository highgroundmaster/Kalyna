  • Byte Wise Diffusion in Kalyna is provided by Mix Columns and Shift Rows and these two operations complement each other
  • Bit Wise Diffusion is provided by Sub Bytes in a non-linear way
    • Sub Bytes works as a 8-bit S-box. Thus if one bit change, the 8 bits of the byte are likely to change as it would map to different number now. With this step, each bit of a byte depend of each other.
  • Together the Shift Rows and Mix Columns operations help to ensure that the number of what are termed active S-boxes is either large to begin with, or becomes large very soon.
  • This forms the basis for the very sound security offered by the Kalyna against differential and linear cryptanalysis.

Shift Row

  • Any pattern or relationship between elements within the same column is disrupted
  • Fairly easy way of creating diffusion

Mix Columns

  • The relationships within a column are difficult to track, elements in the same row will be modiﬁed differently.
  • The Mix Column matrix is derived from the parity-check matrix of a Maximally Distance Separable (MDS) code
  • Thus, if two inputs to the Mix Columns layer have similar Input Differences, the outputs will still be very different and this should help to amplify even a small amount of “change” or “difference” over successive rounds of the cipher.
  • Increased size of MDS matrix when compared to AES provides even better propagation properties, so the strength to the most of cryptanalytical methods.
